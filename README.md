# Decision-Tree
This repo implements Decision Tree [Classification](https://github.com/vamc-stash/Decision-Tree/blob/master/src/DecisionTreeClassifier.py) and [Regression](https://github.com/vamc-stash/Decision-Tree/blob/master/src/DecisionTreeRegressor.py) algorithms from scratch in Python.

Please read the [blog](https://medium.com/@rangavamsi5/decision-tree-implementation-from-scratch-in-python-1cff4c00c71f) that goes with code.
